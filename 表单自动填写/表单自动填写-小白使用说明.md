# 表单自动填写助手小白使用说明

这份说明是写给完全不懂代码、第一次接触油猴脚本的人看的。

你只需要记住一句话：

这个工具的作用，就是你打开指定表单页面后，它会按照你提前写好的内容，自动把表单一项一项填进去。

## 一、先搞懂它能做什么

这个工具适合下面这种页面：

- 你每次都要重复填写同一类表单
- 页面上的输入框顺序基本固定
- 就算页面文字偶尔变了、标题变了，只要输入框顺序没乱，它也还能用

比如：

- 第 1 个框填姓
- 第 2 个框填名
- 第 3 个框填学号
- 第 7 个是单选，选择 `China (Mainland)`

它就会按这个顺序自动填写。

## 二、你需要准备什么

你只需要准备 2 样东西：

1. 一个浏览器
推荐：
- Chrome
- Edge

2. 一个油猴扩展
油猴英文名叫 `Tampermonkey`

简单理解：

油猴就是一个“让浏览器运行小脚本”的工具。

## 三、第一步：安装油猴

### Chrome 安装方法

1. 打开 Chrome 浏览器
2. 去 Chrome 应用商店搜索 `Tampermonkey`
3. 点击安装
4. 安装完成后，浏览器右上角会出现油猴图标

### Edge 安装方法

1. 打开 Edge 浏览器
2. 去 Edge 插件商店搜索 `Tampermonkey`
3. 点击获取 / 安装
4. 安装后，右上角会出现油猴图标

如果你不会找插件商店，也可以直接在搜索引擎搜：

`Tampermonkey Chrome`

或者

`Tampermonkey Edge`

## 四、第二步：安装这个脚本

你当前要安装的脚本文件是：

`表单自动填写.user.js`

### 安装方法

1. 找到这个文件
2. 用浏览器打开它
3. 油猴一般会自动弹出安装页面
4. 点击“安装”

如果没有自动弹出来，可以这样做：

1. 点击浏览器右上角油猴图标
2. 选择“创建新脚本”或者“添加新脚本”
3. 把 `表单自动填写.user.js` 里的全部内容复制进去
4. 保存

保存后，这个脚本就已经装好了。

## 五、第三步：第一次打开脚本配置

这个脚本有 2 种打开配置的方法。

### 方法 1：页面右下角按钮

安装脚本后，随便打开一个网页。

如果脚本正常运行，页面右下角一般会出现一个蓝色小按钮：

`配置`

点它，就能打开设置面板。

### 方法 2：油猴菜单

1. 点击浏览器右上角油猴图标
2. 找到这个脚本
3. 选择：
   - `打开表单配置`
   - `导出 JSON 配置`
   - `导入 JSON 配置`
   - `清空当前配置`

如果你是第一次用，最常用的是：

`打开表单配置`

## 六、第四步：最重要的配置怎么填

打开配置面板后，你会看到几块内容。

### 1. URL 匹配规则

这里填写“脚本在哪个网页自动运行”。

比如你的表单页面地址是：

`https://abc.com/form/123`

那你可以写：

`https://abc.com/form/*`

意思是：

只要打开 `https://abc.com/form/` 开头的页面，脚本就自动运行。

### 2. 自动运行

这里一般保持“开启”。

意思是：

你打开目标页面后，脚本自动开始填写，不需要你手动点开始。

### 3. 提交模式

这里有 2 种：

- `仅填写不提交`
- `填写后自动提交`

给小白的建议：

一开始一定选 `仅填写不提交`

原因很简单：

先让它自动填，填完你自己检查一遍，确认没问题再手动点提交。

等你确认这个脚本稳定了，再改成自动提交。

### 4. 自动提交延时(ms)

这个意思是：

如果你开启了自动提交，填完后等多久再点提交按钮。

一般默认 `1500` 就够了。

`1500` 的意思是 1.5 秒。

### 5. 字段列表

这个是最核心的地方。

你会看到每一行有 4 个重要内容：

- 顺序
- 类型
- 标签
- 预填值

#### 顺序是什么

顺序就是“页面上第几个可填写控件”。

例如一个表单页面：

1. 第 1 个输入框：姓
2. 第 2 个输入框：名
3. 第 3 个输入框：学号
4. 第 4 个输入框：电话
5. 第 5 个输入框：微信
6. 第 6 个输入框：护照号
7. 第 7 个控件：国籍单选

那你就按这个顺序写。

#### 类型怎么选

脚本支持下面几种类型：

- `text`：普通输入框
- `select`：下拉框
- `radio`：单选
- `checkbox`：复选
- `date`：日期

不会判断时，你可以这样记：

- 能直接打字，基本是 `text`
- 点开选一个，基本是 `select`
- 圆圈单选，选一个，基本是 `radio`
- 方框多选，基本是 `checkbox`
- 填日期，基本是 `date`

#### 标签是什么

标签只是方便你自己看懂，不影响程序运行。

比如你可以写：

- 第1项-姓
- 第2项-名
- 第3项-学号

#### 预填值是什么

就是你想自动填进去的内容。

例如：

- 姓：`ZHANG`
- 名：`San`
- 学号：`20250001`

## 七、最简单的配置例子

假设你的页面顺序是：

1. 姓
2. 名
3. 学号
4. 电话
5. 微信
6. 护照号
7. 国籍单选

你可以这样填：

- 第 1 项
  - 顺序：`1`
  - 类型：`text`
  - 标签：`姓`
  - 预填值：`ZHANG`

- 第 2 项
  - 顺序：`2`
  - 类型：`text`
  - 标签：`名`
  - 预填值：`San`

- 第 3 项
  - 顺序：`3`
  - 类型：`text`
  - 标签：`学号`
  - 预填值：`20250001`

- 第 4 项
  - 顺序：`4`
  - 类型：`text`
  - 标签：`电话`
  - 预填值：`13800138000`

- 第 5 项
  - 顺序：`5`
  - 类型：`text`
  - 标签：`微信`
  - 预填值：`weixin123`

- 第 6 项
  - 顺序：`6`
  - 类型：`text`
  - 标签：`护照号`
  - 预填值：`E12345678`

- 第 7 项
  - 顺序：`7`
  - 类型：`radio`
  - 标签：`国籍`
  - 预填值：`China (Mainland)`

填好后，点击：

`保存配置`

就行了。

## 八、以后怎么正式使用

配置一次后，以后使用步骤就很简单：

1. 打开目标表单页面
2. 脚本自动识别这个页面是不是你配置过的网址
3. 如果是，就会自动开始填写
4. 页面右下角会提示：
   - `采集中...`
   - `填写完成`
   - 或 `部分失败`
5. 如果你设置的是“仅填写不提交”，那你检查一下内容，然后自己点提交
6. 如果你设置的是“填写后自动提交”，那它填完后会自己点提交按钮

## 九、导出和导入配置怎么用

这个功能很重要。

因为你以后可能：

- 想备份配置
- 想复制给别人用
- 想换电脑继续用

### 导出配置

方法：

1. 点击油猴菜单
2. 选择 `导出 JSON 配置`
3. 会弹出一大段文字
4. 全部复制下来，保存到记事本里

### 导入配置

方法：

1. 点击油猴菜单
2. 选择 `导入 JSON 配置`
3. 把之前保存的那段文字粘贴进去
4. 确定

这样配置就恢复了。

## 十、什么叫 JSON，你不用害怕

很多人看到 `JSON` 就慌，其实你把它理解成“有格式的配置文本”就行。

比如下面这段：

```json
{
  "pageRule": {
    "urlPattern": "https://example.com/form/*",
    "autoRun": true,
    "submitMode": "manual",
    "submitDelayMs": 1500
  },
  "fields": [
    { "index": 1, "type": "text", "label": "姓", "value": "ZHANG" },
    { "index": 2, "type": "text", "label": "名", "value": "San" },
    { "index": 7, "type": "radio", "label": "国籍", "value": "China (Mainland)" }
  ]
}
```

你不用完全看懂，只需要知道几件事：

- `index` 是顺序
- `type` 是类型
- `label` 是备注
- `value` 是要填的内容

## 十一、单选和复选怎么填

### 单选 radio

例如页面有：

- China (Mainland)
- China (Hong Kong)
- Other

你想选第一个，就写：

`China (Mainland)`

脚本会优先按文字去找。

### 复选 checkbox

如果有多个选项，你可以这样写：

`足球,篮球,游泳`

中间用英文逗号隔开。

脚本会尽量把这些选项都勾上。

## 十二、日期怎么填

日期建议写成这种格式：

`2026-03-12`

也就是：

年-月-日

这是最稳的格式。

## 十三、如果填写失败了怎么办

先别慌，这个脚本已经做了基础防崩处理。

它失败时一般不会把页面弄坏，只会提示你哪里有问题。

### 常见情况 1：页面没反应

先检查：

1. 你打开的网址，是不是和 `URL 匹配规则` 一致
2. 油猴脚本是不是启用了
3. 浏览器右上角油猴图标有没有显示正常

### 常见情况 2：有些内容填了，有些没填

这通常说明：

- 页面控件顺序变了
- 你设置的顺序不对
- 你把类型选错了

比如：

你本来把第 7 项设成 `radio`

但页面第 7 项现在变成了输入框

那脚本就会提示类型不匹配。

解决方法：

重新对照页面，从上往下数一遍顺序，再改配置。

### 常见情况 3：下拉框选不中

可能原因：

- 你填的值和下拉选项文字不一样
- 大小写或空格不一致
- 页面这个下拉不是标准下拉框

解决方法：

直接把页面里显示的文字，一模一样复制进去。

### 常见情况 4：单选没选上

可能原因：

- 你填的文字和页面选项不完全一样
- 页面结构变了

解决方法：

先试文字完全照抄。

如果还不行，可以试数字：

- `1` 表示第一个选项
- `2` 表示第二个选项
- `3` 表示第三个选项

### 常见情况 5：自动提交没生效

可能原因：

- 没开自动提交
- 页面上提交按钮比较特殊，脚本没认出来

这时候先用“仅填写不提交”，人工点提交即可。

## 十四、怎么看日志

如果你愿意稍微进阶一点，可以看浏览器控制台日志。

### 打开方法

在页面按：

- Windows：`F12`
- Mac：`Command + Option + I`

然后点：

`Console`

你会看到一些以这个开头的日志：

`[form-autofill]`

这些日志会告诉你：

- 页面有没有匹配成功
- 找到了多少个控件
- 哪一项填成功了
- 哪一项失败了

如果以后找人帮你排查问题，把这些日志截图发给对方就行。

## 十五、给小白的实战建议

第一次使用时，强烈建议你这样做：

1. 先只配置 2 到 3 个字段测试
2. 提交模式一定选“仅填写不提交”
3. 确认顺序没问题后，再慢慢把剩余字段加上
4. 连续测试 2 到 3 次都正常，再考虑自动提交

不要一上来就直接全自动提交。

这样最安全。

## 十六、哪些情况不适合这个脚本

下面这些情况，首版不一定好用：

- 富文本编辑器
- 图片上传
- 文件上传
- 非常复杂的联动下拉
- 页面内容在 iframe 里面
- 一个页面里有很多块非常复杂的表单混在一起

如果你的页面属于这种情况，就需要后续再专门增强。

## 十七、最常用的一句话版操作流程

如果你已经装好了脚本，平时就记这一套：

1. 打开表单页
2. 点右下角 `配置`
3. 填好网址规则和字段顺序
4. 点 `保存配置`
5. 刷新页面
6. 看它自动填写
7. 检查无误后手动提交

## 十八、最后给你的建议

这个工具最关键的不是代码，而是：

你必须把“页面上第几个控件对应什么内容”这件事数清楚。

只要顺序数对了，这个工具就很好用。

如果顺序数错了，它就会填错地方。

所以第一次配置时，最重要的动作不是着急填，而是：

拿着页面，从上往下，一个一个数。

数对了，后面基本就顺了。
